# React Project Setup Guide

Follow the steps below to run this project on your computer.

## 1. Clone the Repository

```bash
git clone <repository-url>
```

Example:

```bash
git clone git@github.com:bicrimalavimanyu/SEP_WEB.git
```

## 2. Navigate to the Project Folder

```bash
cd SEP_WEB/myapp
```

## 3. Verify You Are in the Project Root

Run:

```bash
ls
```

You should see a `package.json` file.

If `package.json` is present, you are in the correct project directory.

## 4. Install Project Dependencies

```bash
npm install
```

This command downloads and installs all required packages.

## 5. Start the Development Server

```bash
npm start
```

## 6. Open the Application

Once the server starts, open your browser and visit:

```
http://localhost:3000
```

The React application should now be running successfully.
